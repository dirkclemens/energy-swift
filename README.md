# energy-swift
macOS Swift App (MQTT) Energiebilanz 

macOS Swift App using MQTT to show data from some of my energy sources (first approach)

Data sources: 
* Growatt inverter
* go-eCharger Homefix
* OBIS / SML reader (esp8266 based)
